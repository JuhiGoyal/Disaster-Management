# ResQNet: Professional Emergency Response System

This repository contains the complete codebase for **ResQNet**, a comprehensive disaster management and resource coordination platform.

## 🚀 Deployment Instructions

### 1. Fresh Repository Setup
The Git history has been re-initialized. To push this to your new repository:
```bash
git remote add origin <YOUR_NEW_REPO_URL>
git branch -M main
git push -u origin main
```

### 2. Environment Configuration
Ensure your `.env` files (located in the `backend` folder) are configured with:
- `MONGODB_URI`: Your MongoDB connection string.
- `JWT_SECRET`: A secure random string for authentication.
- `EMAIL_USER`: Your support email (e.g., Gmail).
- `EMAIL_PASS`: Your Gmail App Password.

### 3. Project Structure
- **/natural-disastermanagement/backend**: Node.js/Express API.
- **/natural-disastermanagement/frontend**: React.js logic and UI.
